**Vue 的安裝:** <br>
可以有好幾種方式，可以透過 npm,yarn……
我們可以簡單用 CDN 的方式引入就好，就跟在一般專案引入 jQuery 一樣，直接在 html 的 Head 部分加上： <br>
`<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>` <br>
